  • Keep up to date — Automatic tracks installed versions and notifies you when upstream updates are available.

Vercel React Best Practices

Overview

Performance optimization guide for React and Next.js applications, covering component patterns, server components, data fetching, and bundle optimization strategies from Vercel Engineering.

Skill Format

Each reference file follows a hybrid format for fast lookup and deep understanding:

  • *Quick Pattern: Incorrect/Correct code snippets for immediate pattern matching
  • *Quick Command: Shell commands for process/measurement skills
  • *Quick Config: Configuration snippets for setup-focused skills

When to Apply

Reference these guidelines when:

  • *Building or refactoring React components
  • *Implementing data fetching in Next.js
  • *Optimizing bundle size or load performance
  • *Reducing unnecessary re-renders

Configuration Templates

Manage configuration across projects with templates

Stop copy-pasting MCP server configs and skill lists between every new project. Define a template once and apply it everywhere — with per-project overrides when you need them.

  • Bundled templates — group skills, MCP servers, and instructions under a single reusable name like "Laravel API" or "React App".
  • Apply to any project — assign a template and the project inherits its full configuration, synced automatically.
  • Per-project overrides — a project can extend a template and add or override specific settings without breaking the base.
  • Team sharing — export and share templates with teammates so everyone's agent environments start from the same baseline.

Built for the Agentic Era

Automatic doesn't run your agents; it empowers them. It acts as the ultimate sidecar for tools like Claude Code and Cursor.

  • Team Configuration Management: Share your configured MCP servers, skills, and prompts with your entire team. Automatic keeps everyone's local agents in sync.
  • Seamless Project Sync: Push required skills, instructions, and server connections automatically to agent-specific directories when you switch contexts.
  • Universal Compatibility: Exposes a standard MCP Server interface, making it universally plug-and-play with any agent that supports the Model Context Protocol (like Claude Code or Cursor).
