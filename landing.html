Everything your agents need to work.

Manage the context surrounding your coding agents. Built with a lightning-fast Rust core and a modern React interface.

Model Context Protocol

Exposes a unified MCP Server interface using stdio transport. Your agents connect to Nexus to instantly access your entire configured ecosystem.

Skill Management

Create, read, update, and delete Claude Skills with a beautiful UI. Synchronize project-specific skills automatically to the right directories.

Secure Credentials

Manage your LLM API providers securely. Nexus safely handles your keys locally so your automated scripts don't have to keep asking.

Built for the Agentic Era

Nexus doesn't run your agents; it empowers them. It acts as the ultimate sidecar for tools like Claude Code and Cursor.

  • Centralized Secrets Management: Store your API keys in one secure location locally, without needing to copy-paste them into every new project's `.env` file.
  • Seamless Project Sync: Push required skills, instructions, and server connections automatically to agent-specific directories when you switch contexts.
  • Universal Compatibility: Exposes a standard MCP Server interface, making it universally plug-and-play with any agent that supports the Model Context Protocol (like Claude Code or Cursor).
